登录捷智算控制台
================

本章节介绍如何登录捷智算 (Jiezhisuan) 网页控制台。

1. 访问捷智算官网：`https://www.gogpu.cn <https://www.gogpu.cn>`_
2. 点击右上角的 **“登录/注册”** 按钮，或者点击 **“控制台”**。

   .. image:: images/console_button.png
      :alt: 控制台按钮
      :align: center
      :scale: 50%

3. 您可以选择以下方式登录：
    *   **微信扫码**：关注“捷智算云服务”公众号进行登录。
    *   **手机号登录**：使用短信验证码或密码登录。
4. 登录成功后，点击 **“控制台”** 进入管理页面。
5. 在左侧导航栏选择 **“我的实例”**，您将看到您租用的服务器列表。
