登录捷智算控制台
================

本章节介绍如何登录捷智算 (Jiezhisuan) 网页控制台。

1. 访问捷智算官网：`https://www.gogpu.cn <https://www.gogpu.cn>`_
2. 点击右上角的 **“登录/注册”** 按钮，或者点击 **“控制台”**。

   .. image:: images/console_button.png
      :alt: 控制台按钮
      :align: center
      :scale: 50%

3. 在登录页面，点击下方的 **“子账号登录”**。
4. 输入您的子账号和密码进行登录。

   .. image:: images/login_screen.png
      :alt: 子账号登录
      :align: center
      :scale: 50%

5. 登录成功后，点击 **“控制台”** 进入管理页面。
6. 在左侧导航栏选择 **“我的实例”**，您将看到您租用的服务器列表。
