登录捷智算
===========

本章节介绍如何登录捷智算 (Jiezhisuan) 平台及服务器。

控制台登录
----------

1. 访问捷智算官网：`https://www.gogpu.cn <https://www.gogpu.cn>`_
2. 点击右上角的 **“登录/注册”** 按钮，或者点击 **“控制台”**。

   .. image:: _static/console_button.png
      :alt: 控制台按钮
      :align: center
      :scale: 50%

3. 您可以选择以下方式登录：
    *   **微信扫码**：关注“捷智算云服务”公众号进行登录。
    *   **手机号登录**：使用短信验证码或密码登录。
4. 登录成功后，点击 **“控制台”** 进入管理页面。
5. 在左侧导航栏选择 **“我的实例”** (或类似名称)，您将看到您租用的服务器列表。

SSH 登录
--------

在控制台的 **“我的实例”** 列表中，找到您需要连接的实例。

1.  复制实例卡片上的 **SSH 登录指令**。通常格式如下：

    .. code-block:: bash

       ssh username@server_address -p port

    *   **username**: 通常为 `root` 或您创建的用户名。
    *   **server_address**: 服务器的公网 IP 地址。
    *   **port**: 映射的 SSH 端口号 (例如 12345)。

2.  打开您的 SSH 客户端（如 PuTTY, MobaXterm, 或本地终端）。
3.  粘贴并运行复制的命令。
4.  输入密码（在实例详情中查看或重置）即可登录。
