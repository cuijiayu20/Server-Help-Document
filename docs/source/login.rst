登录捷智算
===========

本章节介绍如何登录捷智算服务器。

SSH 登录
--------

使用 SSH 客户端（如 PuTTY, MobaXterm, 或终端）连接到服务器。

.. code-block:: bash

   ssh username@server_address -p port

* **username**: 您的用户名
* **server_address**: 服务器 IP 地址
* **port**: SSH 端口号 (默认 22)

Web 登录
--------

如果服务器提供 Web 控制台，请在浏览器中访问：

http://server_address:port

输入您的用户名和密码进行登录。
