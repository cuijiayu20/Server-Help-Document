租用服务器
============

本章节介绍如何在捷智算平台租用云主机。

租用步骤
--------

1. **进入实例管理**
   
   登录控制台后，在快捷入口选择 **"实例管理"**。

   .. image:: images/instance_management.png
      :alt: 实例管理入口
      :align: center
      :scale: 50%

2. **创建云主机**
   
   点击 **"创建云主机"** 按钮。

   .. image:: images/create_instance.png
      :alt: 创建云主机
      :align: center
      :scale: 50%

3. **选择配置**
   
   根据需求选择合适的主机配置：
   
   * **计费方式**：按小时、包日、包周、包月等
   * **选择地区**：广东、北京、安徽、四川等
   * **GPU 型号**：根据需求选择显卡类型
   * **卡数/规格**：选择单卡、2卡+、4卡+ 等

   .. image:: images/select_instance.png
      :alt: 选择实例配置
      :align: center
      :scale: 50%

4. **确认并创建**
   
   确认配置信息无误后，点击创建按钮完成租用。

重要提示
--------

.. warning::

   **自动续费扣费说明**
   
   只要开启自动续费，无论是否关机，都会按小时扣费。请注意及时关闭不使用的实例。

.. warning::

   **余额不足处理**
   
   余额不足时，信用额度为 -5 元。超出信用额度将释放实例，请及时充值以避免数据丢失。

.. note::

   **存储费用说明**
   
   免费 50 GB 存储空间。超出部分按 0.0004 元/1 小时/GB 计费。
