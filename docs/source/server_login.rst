登录捷智算服务器
================

本章节介绍如何连接和登录您租用的服务器实例。

SSH 登录
--------

SSH 是最常用的远程连接方式。

1.  **获取登录信息**：
    在控制台的 **“我的实例”** 列表中，找到您需要连接的实例。复制实例卡片上的 **SSH 登录指令**。

    通常格式如下：

    .. code-block:: bash

       ssh username@server_address -p port

    *   **username**: 通常为 `root` 或您创建的用户名。
    *   **server_address**: 服务器的公网 IP 地址。
    *   **port**: 映射的 SSH 端口号 (例如 12345)。

2.  **连接服务器**：
    打开您的 SSH 客户端（如 PuTTY, MobaXterm, 或本地终端），粘贴并运行复制的命令。

3.  **输入密码**：
    输入密码（在实例详情中查看或重置）即可登录。

Web 登录
--------

部分镜像或服务（如 JupyterLab）支持通过浏览器直接访问。

1.  **获取访问地址**：
    在控制台实例详情中，查找 **“Jupyter”** 或 **“WebSSH”** 的链接。
    或者，如果您的服务运行在特定端口（如 8888），请使用 `http://server_address:port` 访问。

2.  **登录**：
    在浏览器中打开链接，输入相应的 Token 或密码进行登录。
